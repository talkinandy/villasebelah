## Project Overview

Villa Sebelah is a modern villa showcase website for twin luxury villas in Kuta Lombok. The project consists of a static HTML wireframe demonstrating the site structure and visual design concepts for the villa presentation website.

## Project Structure

This is a **static HTML project** with the following structure:
- `villa-sebelah-wireframe.html` - Main wireframe showcasing the complete website layout
- `render/` - Contains high-quality architectural render images (8 images total)
- `Dinna Andy House Reportt Book_opt.pdf` - Project documentation/reference material

## Architecture

### Website Wireframe Structure
The wireframe demonstrates a modern, minimalist villa showcase website with:

1. **Navigation Bar** - Sticky navigation with villa sections
2. **Hero Section** - Full-screen image slider with call-to-action
3. **Twin Villa Concept** - Showcases both 2+1 bedroom villas side by side
4. **Features Grid** - 8 key features including modern architecture, private pools, flexible spaces
5. **Gallery Preview** - Image grid for architectural renders
6. **Location Section** - Interactive map and location benefits
7. **Contact/CTA Section** - Contact form and booking inquiries
8. **Footer** - Multi-column layout with social links

### Design Principles
- **Minimalist Design**: Clean layouts with white space matching architectural style
- **Visual Hierarchy**: Large, high-quality images as primary focus
- **Twin Villa Focus**: Clear presentation of both identical 2+1 bedroom villas
- **Flexible Configuration**: Highlights 2+1 bedroom layout with convertible spaces
- **Mobile Responsive**: All sections designed to adapt to mobile screens

## Development Commands

This is a static HTML project with no build process. To work with the files:

- **View wireframe**: Open `villa-sebelah-wireframe.html` in any modern web browser
- **No build/test/lint commands**: Pure HTML/CSS project requires no compilation
- **No package.json**: No Node.js dependencies or npm scripts
